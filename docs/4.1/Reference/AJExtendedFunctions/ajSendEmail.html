    ajSendEmail function

    Description

    The ajSendEmail function sends an email to recipients with a given subject, message, and attachments.

    Syntax

    ajSendEmail(subject,message,to_address,[cc_address],[bcc_address],[attachment_path],[html_mail],[smtp_id],[host],[port],[username],[password],[sender_address],[run_condition],[run_by_function_point_only])

    Argument NameArgument TypeDescription
    subject
    (required)    		StringSpecify subject for the email.

    message
    (required)    		RangeSpecify the content of the email from a range of cells. The range of cells can span across multiple rows and columns. Each row represents a broken line and all the columns will be concatenated.

    to_address
    (required)    		RangeSpecify TO email address from a range of cells. The range of cells can span across multiple rows and columns. Each cell should contain one email address.

    cc_address
    (optional)    		RangeSpecify CC email address from a range of cells. The range of cells can span across multiple rows and columns. Each cell should contain one email address.

    bcc_address
    (optional)    		RangeSpecify BCC email address from a range of cells. The range of cells can span across multiple rows and columns. Each cell should contain one email address.

    attachment_path
    (optional)    		RangeSpecify the attachment file paths of the email from a range of cells. The range of cells can span across multiple rows and columns. Each cell contains one file path.

    If you do not specify anything, the default value will always be an empty string (“”), which is no attachment.

    html_mail
    (optional)    		BooleanIf it equals FALSE, the function will process the Body as plain text.

    If it equals TRUE, the function will process the Body as HTML formatted.

    If you do not specify anything, the default value will always be FALSE.

    smtp_id
    (optional)    		StringSpecify the SMTP Connection ID defined in worksheet ##ExternalResources.

    If not provide 'smtp_id', parameter 'host','port', 'sender_address' are mandatory.
    If provide the 'smtp_id' and not provide 'host','port' and 'sender_address', the 'Host','Port', 'Sender Address' columns are mandatory in the SMTP Connection setup.
    If provide the info both in 'smtp_id' and parameter such as 'host','port', 'username','password','sender_address', the parameters value will overwrite the column value provided worksheet ##ExternalResources.

    host
    (optional)    		StringSpecify the IP address or FQDN of the SMTP server.

    port
    (optional)    		StringSpecify the listening port of the email service of the SMTP server.

    username
    (optional)    		StringSpecify the email account login user name.

    password
    (optional)    		StringSpecify the password for the username.

    sender_address
    (optional)    		StringSpecify FROM email address.

    run_condition
    (optional)    		Boolean

    The function will run when the value is TRUE. Otherwise will not run.

    If you do not specify anything, the default value will always be TRUE.

    run_by_function_point_only
    (optional)    		BooleanIf it equals FALSE, the function can be executed through ‘Excel Calculation’ (can be either Automatic or Manual, Calculate Now or Calculate Sheet) or Preview Function Point.

    If it equals TRUE, the function can be executed with Preview Function Point (AlchemyJ toolbar \ Preview Function Point) only.

    If you do not specify anything, the default value will always be TRUE.

    The function will return:

    1) Content type: TRUE / #Value

    2) Method: Within a cell

    Example

    Firstly, setup the STMP Connection and use the STMP Connection ID as 'primary'. The mail server information could be setup in ##ExternalResources or input as function parameters.

    This example sends an HTML email to an email address.

    The result is TRUE which indicates the email has been sent out successfully. However, it does not necessarily mean the recipient can receive or has received the email. Whether a recipient receives the email depends on how the mail server handles the email. That is beyond the reach of this function.

    It will return #VALUE! when there is an error in the usage of the function.

    As Html_mail is TRUE and an email body is in HTML format, the mail client displays the content with HTML formatting.

    attachment_pathAttachment path does not exist.

    messageMessage cannot be empty.

    username
    password    		Username cannot be empty if password is provided.
