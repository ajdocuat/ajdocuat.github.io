    ajDBType function

    Description

    The ajDBType function is used to retrieve the database type based on the specified data source ID. The database type are as follows: Oracle, MySQL, PostgreSQL and MSSQL. Please take note that to run this function from Excel, you would need to create a new data source record in the data source configuration table in External Resources Definition.

    Syntax

    ajDBType(data_source_id)

    Argument NameArgument TypeDescription
    data_source_id
    (Optional)    		StringSpecify the data_source_id. If the data_source_id is not provided, it will use the default data source ID value 'primary' in the Data Source Configuration table.

    The function will return:

    1) Content type: Data source type in String format.

    2) Method: Within a cell

    Example

    In the External Resources Definition, create a new data source record in the Data Source Configuration table.

    As you can see, a new data source record is created as shown above. The default data source ID value is 'primary'. The rest of the columns that are not shown in this screenshot are empty.

    Since the data source ID is 'primary', it is used as the input parameter of the ajDBType function. This function returns the result 'MySQL'.
